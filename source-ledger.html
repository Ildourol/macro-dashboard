5. Trade-Desk Freshness Rules

FACT
A value shown directly from a named source, refreshed PDF, or visible web entry.
INFERENCE
A desk interpretation derived from multiple facts. Useful for regime mapping, never raw data.
STALE RISK
Gamma, 0DTE options flow, live index levels, FX spot, and crude levels require same-session refresh before active futures execution.
EXECUTION RULE
When freshness is uncertain, trade smaller, widen confirmation, and avoid using stale levels as exact stop or breakout triggers.
Dashboard interpretation: refreshed PDFs support strategic and tactical regime analysis. They do not replace live order-book validation. Correct retail futures workflow: use the dashboard to identify regime and crowding, then require live price confirmation at support/resistance before entry.
